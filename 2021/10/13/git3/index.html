使用Github远程仓库

  1. 上传本地仓库到Github

分布式版本系统的最大好处之一是在本地工作完全不需要考虑远程库的存在，也就是有没有联网都可以正常工作，当有网络的时候，再把本地提交推送一下就完成了同步，真是太方便了！

  1. 解除本地库和远程库的绑定关系

$ git remote -v : 查看关联远程库的信息

$ git remote rm origin : 解除本地库和远程origin库的关系

这里只是解除连接关系，并不是删除远程库上的文件；如果要删除文件，需要在Github上删除

  1. 将Github上的远程仓库克隆到本地

$ git clone git@github.com:JasonLiuys/gitskill.git
